# ⚡ NetOrchestra AI v3.0 ⚡
### AI-Powered Autonomous Network Controller & Firewall Security Dashboard

NetOrchestra AI is a lightweight, high-velocity network orchestration engine designed to automate infrastructure diagnostics and access control configurations using natural language processing. Built with a 0% local RAM footprint architecture optimized for low-resource environments.

## 🚀 Core Features
- **Intelligent Intent Parsing (NLP):** Automatically maps English instructions to network actions.
- **Automated Infrastructure Link Analyzer:** Real-time Core and Access layer connectivity diagnostics.
- **Kernel-Level Outbound Firewall Guard:** Dynamic injection and cleanup of Windows Defender Firewall rules via `netsh`.
- **Cyberpunk Tactical Cockpit:** High-end, responsive dark-theme UI connected via FastAPI.

## 🛠️ Tech Stack
- **Backend:** Python 3.x, FastAPI, Uvicorn
- **Frontend:** Tactical HTML5, Cyberpunk Custom CSS, Asynchronous Fetch API
- **Security Interface:** Windows Advanced Firewall CLI Integration
